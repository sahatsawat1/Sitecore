เติมเงิน Easy Pass ผ่าน Krungthai NEXT

  • เปิด Krungthai NEXT
  • เลือก “เติมเงิน”
  • ค้นหา “อีซี่พาส (Easy Pass)“
  • ใส่รหัสซีเรียล นัมเบอร์ กรอกจำนวนเงิน กดถัดไป
  • กดรหัสยืนยัน
  • เติมเงินเรียบร้อย

เติมเงิน Easy Pass ผ่านตู้ ATM/ADM ธนาคารกรุงไทย

  • สอดบัตร ATM แล้วใส่รหัส
  • เลือก “เติมเงินบัตรทางด่วน Easy Pass”
  • เลือก “ประเภทบัญชี” ของบัตร
  • ใส่หมายเลข S/N ด้านหลังบัตร Easy Pass 10 หลัก
  • และเงินที่เติม พร้อมกดถูกต้อง
  • รับใบยืนยันการเติมเงิน

เติมเงิน Easy Pass ผ่าน Internet Banking

  • เข้า Internet Banking Website
  • ใส่ User ID และ Password และ Login
  • เลือก “หน่วยงานที่ต้องการชำระ”
  • เพิ่ม “หน่วยงานที่ต้องการชำระ” โดยคลิก “คมนาคม” แล้วคลิก “การทางพิเศษแห่งประเทศไทย (เติมเงิน Easy Pass)” ตั้งชื่อแทน แล้วใส่หมายเลข S/N ด้านหลังบัตร Easy Pass 10 หลัก
  • ทำรายการเติมเงิน
  • เลือกจ่ายเงินค่า Easy Pass จากชื่อที่เราตั้งไว้
  • เลือกบัญชีที่จะจ่ายเงิน ใส่จำนวนเงินและคลิก ตกลง
  • ยืนยันการจ่ายเงิน
  • เติมเงินสำเร็จ

เติมเงิน Easy Pass ด้วยการผูกบัญชีอัตโนมัติ ผ่าน EXAT Portal Application

  • เปิด EXAT Portal Application
  • เลือก “Easy Pass” แล้วเข้าสู่ระบบ
  • เลือกบัตรที่ต้องการผูกบัญชี
  • เลือกจำนวนเงิน
  • เลือกธนาคารที่ต้องการผูกบัญชี
  • เลือกการยืนยันตัวตน (บัญชี Internet Banking หรือ บัตรเดบิต)
ข้อกำหนดและเงื่อนไข
  • - สามารถเติมเงินได้ตั้งแต่ 300 บาท ถึงสูงสุด 5000 บาท/ครั้ง
  • - การเติมเงินผ่านแอป Krungthai NEXT ไม่มีค่าธรรมเนียม
  • - การเติมเงินผ่านตู้ ATM ค่าธรรมเนียม 15 บาท/รายการ
  • - บริการนี้จำเป็นต้องมีบัตร
  • - เมื่อยอดเงินคงเหลือในบัตร Easy Pass เหลือต่ำกว่า 200 บาท ระบบจะทำการตัดเงินในบัญชีเติมเข้าบัตร Easy Pass ให้ทันที ตามจำนวนเงินที่สมาชิกได้เลือกไว้ขั้นต่ำ (ขั้นต่ำ 300 บาท สูงสุด 5,000 บาท)
  • - หากต้องการรับใบรับอิเล็กทรอนิกส์ (e-Receipt) หรือใบกำกับภาษีอิเล็กทรอนิกส์ (e-Tax Invoice) สามารถลงทะเบียนขอรับบริการ ได้ที่ EXAT E-Service ผ่านทาง Application EXAT Portal หรือที่ www.thaieasypass.com
  • - ระบบจะทำการส่งใบรับอิเล็กทรอนิกส์ (e-Receipt) ไปยังอีเมล์ที่สมาชิกได้ลงทะเบียนไว้ทันที
  • - ระบบจะทำการส่งใบกำกับภาษีอิเล็กทรอนิกส์ (e-Tax Invoice) ตามรอบการจัดส่งรายเดือน (ทุกวันที่ 5 ของเดือน)
  • - สอบถามข้อมูลเพิ่มเติมได้ที่ศูนย์บริการข้อมูลผู้ใช้ทางพิเศษ EXAT Call Center 1543 ได้ตลอด 24 ชั่วโมง
