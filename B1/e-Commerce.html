บริการที่ธนาคารทำหน้าที่เป็นตัวกลางรับชำระเงินค่าสินค้าและบริการที่ลูกค้าสั่งซื้อจากระบบ e-Commerce เช่น amazon.com ผ่านระบบของ

ลักษณะบริการ

  • แจ้งรหัสชำระเงิน Pay Code หรือรหัสขอคืนเงิน (Refund Code) ที่ได้รับจากการสั่งซื้อสินค้า/บริการบน e-Commerce มาชำระเงินสดที่ธนาคาร
  • ฟรี! ไม่มีค่าธรรมเนียม (สกุลเงินบาทเท่านั้น)
  • ผู้ทำธุรกรรมต้องเป็นบุคคลเดียวกับผู้ซื้อสินค้า/บริการจากระบบ e-Commerce โดยต้องแสดงบัตรประชาชนหรือพาสปอร์ต

ประเภทของบริการ

  • การรับชำระเงินค่าสินค้า online (Payment)
  • การคืนเงินค่าสินค้า (Refund)

ข้อควรรู้

  • ธนาคารสงวนสิทธิ์ในการปฎิเสษให้บริการในกรณีที่ลูกค้าไม่ได้ทำรายการด้วยตัวเอง และ/หรือจำนวนเงินที่ทำรายการไม่ถูกต้องตามที่ปรากฎในระบบ และ/หรือกรณีที่ Pay Code หรือ Refund Code ที่แจ้งไม่ถูกต้องหรือหมดอายุ
  • หากมีข้อสงสัยสามารถดูได้จาก www.amazon.com/paycode/help
